# 009: 写真アップロード（クライアントリサイズ + プレビュー）

> 優先度: 高 | 依存: 007

## 参照

- `docs/要件定義書.md` セクション 4.5.5（写真アップロード仕様）
- `docs/デザインガイドライン.md` セクション 5.4（ファイルアップロード UI）

## やること

### PhotoUpload コンポーネント

- [ ] ドラッグ＆ドロップゾーン
  - 破線ボーダー `#E5E5E5`、中央にアイコン + テキスト
  - ドラッグオーバー時のビジュアルフィードバック
- [ ] ファイル選択ボタン（クリックでも選択可能）
- [ ] 対応フォーマット: jpg, png, heic, webp
- [ ] ファイル数上限: 約10枚（ソフトリミット）
- [ ] ファイルサイズ上限: 各20MB（ソフトリミット）

### クライアントサイドリサイズ

- [ ] アップロード前に長辺 2000px にリサイズ
- [ ] Canvas API を使用した画像リサイズ処理
- [ ] HEIC → JPEG 変換対応（heic2any ライブラリ等）
- [ ] EXIF orientation 補正

### プレビュー表示

- [ ] アップロード予定ファイルのサムネイルプレビュー
- [ ] 各ファイルの削除ボタン
- [ ] ファイルサイズ表示
- [ ] アップロード進捗表示（送信時）
