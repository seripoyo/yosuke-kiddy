## やること

### Form コンポーネント（メイン）

- [x] SectionHeading: `MESSAGE` / `ご記帳`
- [x] フォーム全体の max-width 600px + center 配置

### フォームフィールド実装

- [x] **ご芳名**（テキスト / 必須）
  - border 1px `#E5E5E5` / focus border `#a06a8c` / 48px height / 2px radius
- [x] **参列のご希望**（マルチチェックボックス / 1つ以上必須）
  - 5つの選択肢をカスタムチェックボックス（24×24px, `#a06a8c`）で表示
  - 選択肢: 通夜に参列 / 告別式に参列 / 通夜・告別式ともに参加 / 自宅焼香 / メッセージのみ
- [x] **郵便番号**（テキスト / 条件付き必須）
  - 式参列を選んだ場合のみ表示
  - 7桁入力、自動ハイフン挿入（120-0036 形式）
- [x] **ご住所**（テキスト / 条件付き必須）
  - 式参列を選んだ場合のみ表示
  - 住所オートコンプリートは 008 で実装
- [x] **建物名・部屋番号**（テキスト / 任意）
- [x] **陽介とのご関係**（テキスト / 任意）
- [x] **本人から呼ばれていたあだ名**（テキスト / 任意）
- [x] **陽介へのメッセージ**（テキストエリア / 任意）
  - min-height 160px, vertical resize
  - 説明文: 「火葬前までに印刷してお棺に入れます」
- [x] **思い出の写真**（ファイルアップロード / 任意）
  - スロットのみ用意（009で実装）

### 条件付き表示ロジック

- [x] 「通夜に参列」「告別式に参列」「通夜・告別式ともに参加」のいずれかが選択された場合 → 郵便番号 + 住所 + 建物名フィールドを表示
- [x] 選択解除時にフィールドを非表示

### バリデーション

- [x] クライアントサイドバリデーション
  - ご芳名: 必須チェック
  - 参列希望: 1つ以上選択チェック
  - 郵便番号: 7桁数字チェック（表示時のみ）
  - 住所: 必須チェック（表示時のみ）
- [x] エラー表示: ボーダー `#C62828` + ⚠アイコン + 3部構成メッセージ
- [x] フォームクローズ対応: `NEXT_PUBLIC_FORM_CLOSED=true` で送信不可表示

### 送信ボタン

- [x] Primary ボタン「送信する ▶」（width: 100%）
- [x] 送信中はローディング状態表示（「送信しています...」）+ disabled で二重送信防止

### 送信完了（簡易版）

- [x] 送信成功時にフォームを完了画面に切り替え
- [x] showLineQR フラグで LINE 表示分岐（詳細は012で実装）
