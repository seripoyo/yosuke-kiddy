### PhotoUpload コンポーネント

- [x] ドラッグ＆ドロップゾーン

  - ドラッグオーバー時のビジュアルフィードバック（pressed カラー）
- [x] ファイル選択ボタン（クリック/タップでも選択可能）
- [x] 対応フォーマット: jpg, png, heic, webp

- [x] EXIF orientation 補正（Canvas API が自動処理）

- [x] ファイルサイズ表示（リサイズ後のサイズ）
- [x] 処理中スピナー表示
